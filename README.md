# Go 抖音去水印工具 & 视频帧提取

基于 Golang 开发的命令行工具，支持抖音等平台的视频去水印处理，并可将视频按帧提取为高清图片。

---

- ✅ 支持抖音视频去水印处理
- 🎞️ 支持将视频按帧提取为图片
- ⚡ 快速、轻量，使用 FFmpeg 实现底层处理
- 🛠️ 支持命令行批量处理

---
